InfoLoker

InfoLoker is a job information website built using the Laravel framework with CRUD (Create, Read, Update, Delete) functionality. The system allows admins to manage job listings efficiently, while users can browse, search, and view detailed job information. This project aims to simplify job posting and job-seeking processes through a structured and responsive web platform.

CRUD Golang Backend

This project is a CRUD backend application developed using Go (Golang) and PostgreSQL. It provides RESTful API endpoints to create, read, update, and delete data, tested and documented using Postman. The system demonstrates efficient backend development practices with secure database interaction, structured routing, and JSON-based API responses.

Employee fatigue prediction

Employee Fatigue Level Prediction System is an intelligent system that utilizes Fuzzy Logic with the Mamdani method to predict employee fatigue levels based on working hours and rest duration. This project aims to assist management in monitoring and preventing excessive work fatigue through a smart and data-driven approach.
